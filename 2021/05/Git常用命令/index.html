Git常用命令

创建版本库

  1. 克隆远程版本库
    git clone &lt;url&gt;
  2. 初始化本地版本库
    git init

    修改和提交

  3. 查看状态
    git status
  4. 查看变更内容
    git diff
  5. 跟踪所有改动过的文件
    git add .
  6. 跟踪指定的问题件
    git add &lt;file&gt;
  7. 文件改名
    git mv &lt;old&gt; &lt;new&gt;
  8. 删除文件
    git rm &lt;file&gt;
  9. 停止跟踪文件但不删除
    git rm --cached &lt;file&gt;
  10. 提交所有更新过的文件
    git commit -m &quot;commit message&quot;
  11. 修改最后一次提交
    git commit --amend

    查看提交历史

  12. 查看提交历史
    git log
  13. 查看指定文件的提交历史
    git log -p &lt;file&gt;
  14. 以列表方式查看指定文件
    git blame &lt;file&gt;

    撤销

  15. 撤销工作目录中所有未提交文件的修改内容
    git rest --hard HEAD
  16. 修改指定的未提交文件的修改内容
    git checkout HEAD &lt;file&gt;
  17. 撤销指定的提交
    git revert &lt;cimmit&gt;

    分支与标签

  18. 显示所有本地分支
    git branch
  19. 切换到指定分支或标签
    git checkout &lt;branch/tag&gt;
  20. 创建新分支
    git branch &lt;new branch&gt;
  21. 删除本地分支
    git branch -d &lt;branch&gt;
  22. 列出所有本地分支
    git tag
  23. 基于最新提交创建标签
    git tag &lt;tagname&gt;
  24. 删除标签
    git tag -d &lt;tagname&gt;

    合并与衍合

  25. 合并指定分支到当前分支
    git merge &lt;branch&gt;
  26. 衍合指定分支到当前分支
    git rebase &lt;branch&gt;

    远程操作

  27. 查看远程版本库信息
    git remote -v
  28. 查看指定远程版本库信息
    git remote shellow &lt;remote&gt;
  29. 添加远程版本库信息
    git remote add &lt;remote&gt; &lt;url&gt;
  30. 从远程库获代码
    git fetch &lt;remote&gt;
  31. 下载代码及快速合并
    git pull &lt;remote&gt; &lt;brabch&gt;
  32. 上传代码及快速合并
    git push &lt;remote&gt; &lt;branch&gt;
  33. 删除远程分支或标签
    git push &lt;remote&gt; :&lt;branch/tag-namr&gt;
  34. 上传所有标签
    git push --tags
